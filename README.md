# ML-UHI-Prediction-Project
Machine Learning project for predicting Urban Heat Island using remote sensing and OSM data.

Project Machine Learning untuk memprediksi Urban Heat Island (UHI) di area perkotaan menggunakan data:
- Landsat 8/9
- OpenStreetMap
- SRTM (elevasi)
- Data cuaca (jika ada)

Folder Structure:
- data_raw/ → data mentah
- data_processed/ → data yang sudah dibersihkan
- notebooks/ → file .ipynb untuk eksperimen
- src/ → script python modular
- models/ → hasil training model
- results/ → evaluasi model
- docs/ → laporan & lampiran
- figures/ → grafik & visualisasi
